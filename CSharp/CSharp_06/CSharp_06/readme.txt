Данное решение содержит 2 проекта:
1) Транслятор кода
2) Абстрактная фабрика 

1) Проект транслятор кода включает в себя:
 1.1 Интерфейс IConvertible, указывающий, что реализующий его класс может конвертировать блок кода в С# или VB код. 
 1.2 Класс ProgramConverter, реализующий интерфейс IConvertible.
 1.3 Интерфейс ICodeChecker с методом CheckCodeSyntax, принимающий две строки: строка для проверки и используемый язык.
 1.4 Класс ProgramHelper, наследемый от класса ProgramConverter и интерфейса ICodeChecker.
 1.5 Консольное приложение, использующее созданные типы.

2) Проект абстрактная фабрика реализует иерархию классов для области кухонной офисной мебели на основе заданной UML-диаграммы классов.
