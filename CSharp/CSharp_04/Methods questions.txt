1) Что такое метод ?
 Это кусок кода, который выполняет какие то действия.
 Методы могут быть статическими, виртуальными и невирутальными.
 Специализированные методы: конструктор, метод расширения, деструктор, индексатор, свойство, событие.

2) Как объявляется метод у экземпляра класса?
 Для объявления метода нужно указать модификатор доступа, возвращаемый тип или void, если метод ничего не возвращает, название метода, в скобках принимаемые параметры.
 Вызывать методы можно можно в соответсвии с модификатором доступа.
 Методы именуются в соотвествии с правилами Pascal Case.
 В скобках после названия метода можно указать параметры, передаваемые в метод.
 Вернуть данные из метода можно с помощью оператора return.

3) Как вызвать метод?
 Чтобы вызвать метод нужно указать имя метода и передать ему параметры.

4) Что такое "Стек вызовов", "стековый фрейм"?
 Стек вызывов - это LIFO стэк, хранящий информацию о том, куда функция должна передать управление после своего завершения.
 Стековый фрейм - одна ячейка стека вызывов, где хранятся локальные данные функции, аргументы, адрес возврата и т.д.
 IL - используется в языках .NET для генерации машинно-независимого кода как результат компиляциии сходного кода,написанного на любом языке программирования.NET.
 Вычислительный стек - мост между приложением и ячейками памяти. Его можно использовать для просмотра параметров функции, локальных переменных, временных объектов.
 Он работает так - сначала переменные и параметры загружаются из памяти в вычислительный стек, затем производятся вычисления, затем результаты помещаются в память из вычислительного стека.
 Вызовы методов на языке IL:вызов статическогометода(call), вызов виртуального метода(callvirt).

5)Как передаются данные в метод и получаются из него?
 Передаваться и получаться данные могут по значению и по ссылке
 При передаче в метод значимых типов по значению создается копия этого типа. В функции производятся операции над копией переменной. После завершения функции переменная переданная в функцию не меняется.
 При передаче в метод ссылочных типов по значению создается копия ссылки этого типа. В функции производятся операции над копией ссылки. После завершения функции переменная переданная в функцию меняется.
 При передаче паметров по ссылке создается косвенная ссылка налокальную переменную, при изменеини ссылки меняется и сам объект как у значимых так и у ссылочных типов. 
 Это позволяет менять переданную ссылку (а не состояние объекта, на который она указывает) внутри метода.
 Выходные параметры являются дополнительной возможностью к возвращению значения из метода. Они помечаются модификатором out и передаются по ссылке.

6)Как метод может принять произвольное количество параметров?
 метод может принимать массив параметров с ключевым словом params.

7) Как объявить метод у самого класса, а не у экземпляра класса?
 Метод нужно пометить модификатором static.
 У самого класса могут быть специализированные методы: конструктор, перегруженные операции, методы расширения.

8) Как можно сделать метод, так чтобы он вызывался "как бы" у объектов существующего класса?
 Нужно не помечать его модификатором static.

9) Можно перегружать операции? 
 Для перегрузки операций нужно определить метод с таким же названием но отличной сигнатурой.
