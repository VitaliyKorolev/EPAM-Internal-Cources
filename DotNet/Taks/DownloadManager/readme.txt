Данное приложение, скачивает файлы из интернета.
Оно может одновременно скачивать несколько файлов, файл скачивать в "несколько потоков".
Для скачивания файла в несколько потоков на сервер направляются Range Requests. Каждый объект Task скачивает свой 
пакет файла (Chunk) и пишет данные в файл в папку C\User\Downloads. Размер пакета может быть 0.5 Мб, 1 Мб, или 2Мб, в зависимости от суммарного 
количества пакетов скачиваемого файла. Количество одновременно выполняющихся Task можно настраивать путем изменения parallelTasksNumber в 
файле конфигурации App.Config.

Для корректной работы приложения, на каждую новую загрузку необходимо создавать новый объект класса DownloadManager.
